This project is an automated solution for evaluating answer sheets, leveraging AI/ML to assess and score student responses. It streamlines the grading process, reducing manual effort and ensuring consistency.
Technologies Used
Python
Scikit-learn (for machine learning models)
OpenCV (for image processing)
Flask (for web deployment)
Setup Instructions
Clone the repository to your local machine.
Install dependencies via pip install -r requirements.txt.
Prepare answer sheet images (e.g., scanned PDFs converted to images).
Run the Flask app with python app.py.
Upload answer sheets and review automated scores and feedback.
Future Improvements
Integrate more advanced NLP models for subjective answers.
Add role-based access for multiple evaluators.
